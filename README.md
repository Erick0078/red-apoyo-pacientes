# Red de Apoyo a Pacientes con Enfermedades de Alto Costo

Proyecto de **Diseño e Implementación de Software** desarrollado en **Java Web**, que busca conectar pacientes con enfermedades de alto costo con donantes, familiares y médicos a través de una plataforma segura y transparente.

---

## Objetivo del Proyecto

Brindar un espacio digital donde pacientes puedan crear campañas de apoyo económico o en especie, validadas por profesionales médicos, y donantes puedan contribuir de manera segura y confiable.

## Principales Funcionalidades

- Gestión de pacientes, médicos, familiares y donantes.  
- Creación, validación y seguimiento de campañas.  
- Donaciones en línea seguras.  
- Comunicación entre usuarios mediante mensajes y notificaciones.  
- Reportes de transparencia y control administrativo.

---

## Tecnologías Utilizadas

- **Lenguaje:** Java (JSP / Servlets)  
- **Base de Datos:** PostgreSQL y MongoDB  
- **Framework:** Spring (opcional)  
- **Servidor:** Apache Tomcat  
- **IDE:** NetBeans / IntelliJ IDEA  
- **Control de Versiones:** Git y GitHub
